字段	含义	示例值
status	状态码，1 表示成功，0 表示失败	1
is_available	域名是否可注册（1=可注册，0=已注册）	0
domain	完整域名	whoiscx.com
domain_suffix	域名后缀	com
query_time	查询时间	2025-06-20 06:06:32
registrant_name	注册人姓名（如未公开则为空）	（空）
registrant_email	注册人邮箱（如未公开则为空）	（空）
registrar_name	注册商名称	Alibaba Cloud Computing Ltd. d/b/a HiChina
creation_time	域名创建时间	2012-04-25 12:36:40
expiration_time	域名到期时间	2026-04-25 12:36:40
creation_days	域名已注册天数	4803
valid_days	域名距离到期剩余天数	309
is_expire	是否已过期（1=已过期，0=未过期）	0
domain_status	域名状态（如正常、锁定等）	ok（正常）
name_server	域名的 DNS 服务器	DNS25.HICHINA.COM
DNS26.HICHINA.COM
whois_server	提供 Whois 信息的服务器地址	grs-whois.hichina.com
raw	原始 Whois 返回内容	（完整原始文本，略）
